### Cómo usar estas fichas
- Empieza por la ficha del dominio.
- Si necesitas ver auth/CSRF por ruta, baja a `../SYSTEM_ROUTES_MATRIX.md`.
- Si necesitas el inventario completo, usa `../SYSTEM_ROUTES_ENDPOINTS.md`.
- Si quieres requests listos, usa `../pronto-api/POSTMAN_USAGE.md` o `../pronto-api/INSOMNIA_USAGE.md`.
- Si necesitas cookies/headers/CSRF por superficie, cruza con `../contracts/pronto-api/domain_contracts.md` y los paquetes `pronto-client` / `pronto-employees` cuando aplique.
