## Contratos públicos de `pronto-api`

### Objetivo
Este directorio concentra los contratos públicos y operativos principales de `pronto-api`.

### Documentos disponibles
- `openapi.yaml` — spec manual priorizada y evolutiva
- `db_schema.sql` — huella estructural disponible/generada para la superficie
- `cookies.md` — cookies observadas y su propósito
- `csrf.md` — canon y excepciones de CSRF
- `headers.md` — headers canónicos de integración
- `domain_contracts.md` — resumen por dominio de auth/cookies/csrf/headers
- `examples.md` — ejemplos mínimos request/response por dominio
- `files.md` — archivos y artefactos asociados al servicio
- `events.md` — eventos y artefactos operativos relevantes
- `redis-keys.md` — namespaces Redis observados para la superficie
